# Intelligent File Carving & Steganography Detection Framework

## Overview
A Python-based digital forensics framework that recovers WAV and GIF files from raw disk images and performs automated steganography detection on recovered audio files.

## Features
- Signature-based WAV and GIF file carving
- Basic fragmented WAV recovery
- Automated LSB steganography detection
- Recovery performance evaluation
- Digital forensic analysis workflow

## Technologies
- Python
- Digital Forensics
- File Carving
- Steganography Analysis

## Usage

```bash
python file_carver_stego_detector.py
```

Enter the path to the forensic image when prompted.
